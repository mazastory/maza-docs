**버전:** 2.0 (상세 확장판) | **작성일:** 2026년 7월 | **기밀 등급:** 대외비

## 1. Executive Summary (사업 개요)

**마자 스튜디오(MAZA Studio)**는 AI 기반 블로그 자동화 SaaS 플랫폼입니다.
코딩·서버·글쓰기 지식이 전혀 없는 비전문가도 클릭 3번만으로 구글 애드센스 최적화 블로그를 개설하고, AI가 24시간 자동으로 고품질 콘텐츠를 **10개 언어**로 동시 발행하여 **달러 광고 수익**을 창출하는 세계 최초의 Zero-IT 다국어 블로그 자동화 플랫폼입니다.

| 항목 | 내용 |
|---|---|
| **사업 유형** | B2C SaaS (구독형) + B2B Agency 에이전시 서비스 |
| **핵심 가치** | ZERO-IT · ZERO-WORK · ZERO-LIMIT |
| **목표 시장** | 부업·재테크에 관심 있는 직장인 / 소상공인 / IT 취약계층 / 시니어 |
| **수익 모델** | 월정액 구독 + 프리미엄 블로그 제작 대행 |
| **초기 투자** | 와디즈 크라우드펀딩 (Pre-Sale) |

대한민국 직장인의 **68%**가 부업에 관심이 있으나, 기존 블로그 수익화는 3가지 장벽으로 인해 **3개월 내 90%가 포기**합니다.

|---|---|---|
| **기술 장벽** | 서버·도메인·CMS 세팅에 수일~수주 소요 | 클릭 3번, 3분 자동 완성 |

| 🌐 **MAZA Zero-IT Blog** | 수익 최적화 최전선 | 구글 SEO 무결점 + 0.1초 초고속 로딩 + 글로벌 다국어 자동화가 기본 탑재된 궁극의 독립형 프리미엄 블로그 |

### 4.1 마자 스튜디오 vs 워드프레스 — 완전 분석

워드프레스는 세계 점유율 1위 CMS이지만, **다국어 블로그 수익화**라는 관점에서는 치명적 한계를 가집니다.

| 비교 항목 | 워드프레스 | 마자 스튜디오 |
|---|---|---|
| **초기 세팅 시간** | 수일 ~ 수주 (도메인·호스팅·플러그인 설정) | **3분** (클릭 3번 자동 완성) |
| **월 운영 비용** | 호스팅 $10~50 + 플러그인 $30~150 = **월 $40~200** | **구독료 포함 올인원** (서버비 0원) |
| **다국어 플러그인** | WPML 연 $99 + 번역 크레딧 종량제 **이중 과금** | **다국어 엔진 내장, 10개 언어 추가비용 0원** |
| **번역 방식** | DeepL/Google API 글자수 × 언어수 = 비용 폭탄 | AI가 원어민 현지화 글 직접 작성 |
| **로딩 속도** | 플러그인 많을수록 3~8초 (SEO 치명타) | **0.1초 (SSR+CDN)** |
| **서버 비용** | 트래픽 증가 시 고사양 업그레이드 강제 | **글로벌 CDN 무료, 트래픽 폭주해도 0원** |
| **SEO 자동화** | 별도 Yoast SEO 플러그인 필요 | **E-E-A-T 최적화 자동 내장** |
| **애드센스 연동** | 수동 코드 삽입 | **자동 삽입 + 검증** |
| **기술 지식 요구도** | 중~고급 (PHP, MySQL 기초 필요) | **ZERO (완전 무지식 OK)** |
| **블로그 수 확장** | 추가 도메인마다 추가 호스팅 비용 발생 | **대시보드에서 원클릭 추가** |
| **자동 발행** | 별도 플러그인 + 설정 복잡 | **오토파일럿 원스위치** |
| **A/S 지원** | 자체 해결 또는 유료 개발자 고용 | **1:1 전담 매니저** |

> **결론:** 워드프레스로 다국어 블로그를 제대로 운영하려면 **초기 세팅비 + 월 $40~200의 운영비 + 기술 습득 시간**이 필요합니다. MAZA는 이 모든 것을 0으로 만듭니다.
