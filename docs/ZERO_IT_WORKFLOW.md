# ⚡ ZeroIT: 원클릭 블로그 세팅 워크플로우

ZeroIT(제로 아이티) 기능을 이용하시면 복잡한 코딩이나 인프라 지식 없이도 마법처럼 블로그 세팅이 완료됩니다. 기존의 수동 세팅 과정(코드 다운로드 ➡️ 깃허브 업로드 ➡️ 넷리파이 연결)을 모두 생략할 수 있습니다.

---

## 🚀 1. Maza Studio 내에서 버튼 3번 클릭으로 해결

Maza Studio의 '새 사이트 추가' 또는 'ZeroIT' 메뉴에서 다음 3단계만 거치면 됩니다.

1. **GitHub 연동 버튼 클릭**
   - 깃허브 로그인 창이 뜨면 권한 승인만 누릅니다.
   - *결과: Maza Blog 소스 코드가 개인 저장소에 자동으로 복사(Fork)됩니다.*

2. **Netlify 연동 버튼 클릭**
   - 넷리파이 로그인 창이 뜨면 권한 승인만 누릅니다.
   - *결과: 넷리파이 서버와 깃허브 코드가 알아서 연결되고 자동 배포 설정이 완료됩니다.*

3. **도메인 입력 및 배포 클릭**
   - 구매하신 도메인(예: `myblog.com`)을 텍스트 칸에 적고 **[배포]** 버튼을 누릅니다.
   - *결과: 이 시점에 Maza Studio 사이트 목록에 내 도메인이 자동 등록되고 서버 호스팅이 시작됩니다.*

---

## 🌐 2. 딱 하나, 외부에서 해야 할 일 (DNS 설정)

위 3단계를 마쳤다면 코딩 없이 사이트가 띄워진 상태입니다. 마지막으로 구매하신 도메인 주소로 접속이 가능하게끔 이름표를 달아주는 작업만 필요합니다.

- **도메인 구입처(가비아, 호스팅케이알 등) 접속**
- **DNS 레코드 설정 메뉴로 이동**
- **A 레코드와 CNAME 레코드 추가:** Netlify의 고정 주소(IP `75.2.60.5` 등)를 향하도록 딱 두 줄만 입력해 주시면 됩니다.
> 👉 상세한 입력 방법은 `domain_connection_guide.md` 문서를 참고하세요.

---

## 💡 요약 및 결론

ZeroIT를 쓰시면 **"코드를 다운받아서 올리고 넷리파이에서 세팅하는 모든 과정"을 마자 스튜디오 안에서 버튼 3번 클릭으로 스킵**할 수 있습니다.

단지 마지막에 외부에서 도메인에 '이름표(DNS)'를 달아주는 작업만 한 번 직접 해주신 뒤, 곧바로 Niche Hunter로 가서 글을 생성하고 **'승인(Approved)'** 처리만 하시면 모든 블로그 운영이 끝납니다!
