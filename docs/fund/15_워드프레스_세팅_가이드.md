# 🔵 워드프레스(WordPress) 원클릭 세팅 가이드

이미 구축해 둔 워드프레스 블로그가 있으신가요? 마자 스튜디오는 워드프레스 REST API 및 Application Password를 활용하여 완벽한 다이렉트 자동 발행을 지원합니다. 별도의 복잡한 플러그인 없이 오직 기본 기능만으로 연동됩니다!

---

## 🛠️ 워드프레스 자동화 마법사 진행 순서

### Step 1. 사이트 정보 및 인증 정보 입력
- **[내 사이트]** 메뉴에서 **[+ 새 사이트 추가]** 버튼을 눌러 마법사를 시작합니다.
- **플랫폼 선택**: `워드프레스 (WordPress)`를 선택합니다.
- **도메인 입력**: 워드프레스 사이트 주소(예: `mywpblog.com`)를 입력합니다.
- **관리자 인증 (매우 중요!)**: 워드프레스 관리자 아이디(Username)와 **앱 비밀번호(Application Password)**를 입력합니다.
  > *팁: 워드프레스 관리자 패스워드가 아닌, 워드프레스 관리자 화면 [사용자] -> [프로필] 하단에서 발급받은 '응용 프로그램 비밀번호(App Password)'를 넣어야 보안상 안전합니다.*

### Step 2. 발행 간격 설정
- 워드프레스 호스팅 서버(예: 클라우드웨이즈, 카페24 등)의 트래픽 과부하를 막기 위해, 하루에 몇 개의 글을 텀을 두고 올릴지 간격을 설정합니다.

### Step 3. 블루프린트(테마) 선택
- 어떤 수익형 주제로 블로그를 운영할지 갤러리에서 **블루프린트**를 선택합니다.

### Step 4. 대량 포스팅 일괄 생성 및 다이렉트 꽂기 🌟
- 생성할 **초기 글의 개수(예: 15개)**를 설정하고 대량 생산을 시작합니다.
- 💡 **[비밀 무기: Backdating]**: AI가 글을 쓸 때 오늘 날짜로만 15개를 올리는 것이 아닙니다. 애드센스 승인을 위해 **'3일 전 5개, 2일 전 5개, 오늘 5개'** 식으로 과거 날짜로 백데이팅하여 워드프레스 DB로 다이렉트로 꽂아버립니다. 유저는 아무것도 할 필요가 없습니다!

### Step 5. 구글 계정 연동 및 메타데이터 세팅
- AI가 뒤에서 글을 워드프레스에 밀어넣는 동안, 화면 안내에 따라 **구글 계정으로 로그인**합니다.
- 시스템이 블로그 검색 노출을 위한 **구글 서치콘솔**과 **GA4(구글 애널리틱스)** 권한을 받아와 워드프레스 헤더에 주입할 메타 태그를 세팅합니다.

### Step 6. 최종 핑(Ping) 전송 및 애드센스 신청
- 글 발행이 완료되면 시스템이 구글 검색엔진에 핑(Ping)을 날려 글 수집을 요청합니다.
- 이제 워드프레스 도메인으로 **구글 애드센스에 승인 심사를 신청**하세요!
