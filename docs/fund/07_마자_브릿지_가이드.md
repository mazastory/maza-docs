# 🚀 MAZA Bridge 스마트 컨텍스트 100% 활용 가이드

웹서핑을 하다가 **"아, 이 내용 내 블로그에 쓰고 싶은데?"** 라는 생각이 든 적 있으신가요? 
이제 복사해서 붙여넣고, AI에게 프롬프트를 입력할 필요가 없습니다. 

**MAZA Bridge 스마트 컨텍스트**가 여러분이 보고 있는 화면을 실시간으로 인식하여 **가장 필요한 글쓰기 액션**을 제안해 줍니다. 

---

## 🎯 MAZA Bridge 스마트 컨텍스트란?

MAZA Bridge(마자 브릿지) 크롬 확장 프로그램이 설치된 상태로 유튜브, 블로그, 뉴스 기사 등을 보다가 **MAZA Studio 탭으로 돌아오면**, 방금 전까지 보고 있던 화면을 분석하여 화면 우측에 **[MAZA Bridge 맞춤형 액션 패널]**을 띄워줍니다. 

사용자는 단 한 번의 클릭만으로 해당 콘텐츠를 내 블로그의 새로운 포스팅으로 재창조할 수 있습니다.

---

## 💡 상황별 활용 방법

어떤 사이트를 보고 있었느냐에 따라 제안되는 버튼이 찰떡같이 달라집니다!

### 📺 1. 유튜브 영상을 보고 있었을 때
유튜브에서 좋은 정보성 영상을 발견했다면? MAZA Studio 탭으로 돌아오세요!
* **✨ 영상 요약하기**: 영상의 핵심 내용만 빠르고 간결하게 요약해 줍니다.
* **🖋️ 스크립트 추출 후 글쓰기 (PRO)**: 영상 전체 스크립트를 AI가 쫙 뽑아내서, 아주 자세하고 퀄리티 높은 정보성 블로그 포스팅으로 완벽하게 탈바꿈시켜 줍니다.
* **⚡ 이 주제로 니치 글쓰기**: 영상의 세부 내용이 아닌 **'주제'**만 차용하여, 완전히 새로운 나만의 수익형 니치 블로그 글을 생성합니다.

### 📝 2. 블로그나 뉴스 기사를 보고 있었을 때
티스토리, 네이버 블로그, 브런치, 뉴스 사이트 등에서 참고할 만한 글을 읽고 있었다면?
* **✨ 핵심 내용 요약**: 긴 기사나 글의 핵심을 빠르게 파악할 수 있도록 3줄 요약해 줍니다.
* **🔖 참고 자료로 저장**: 이 글을 나중에 글쓸 때 참고하기 위해 **[아이디어 금고]**에 원클릭으로 스크랩합니다.
* **🖋️ AI 리라이팅 (PRO)**: 원본 글의 내용을 바탕으로 어투를 변경하고 살을 붙여, 중복 문서(유사 문서)에 걸리지 않는 완전히 새로운 포스팅으로 재작성해 줍니다.

### 🖼️ 3. 이미지를 보고 있었을 때
웹에서 멋진 이미지 파일(단독 페이지)을 보고 있었다면?
* **🔖 이미지 마자스튜디오로 보내기**: 해당 이미지를 내 아이디어 금고에 즉시 저장합니다.
* **🖼️ 글 섬네일로 지정 (PRO)**: 현재 작성 중인 블로그 글의 대표 이미지(썸네일)로 쏙 집어넣습니다.
* **✨ AI 이미지 설명 생성 (PRO)**: 이미지를 AI가 분석하여, 이 이미지를 설명하는 글을 자동으로 써줍니다.

### 🔍 4. 구글/네이버에서 검색 중이었을 때
포털 사이트에서 특정 키워드를 검색한 결과 창을 보고 있었다면?
* **🖋️ 이 키워드로 니치 글쓰기**: 당신이 검색했던 그 키워드를 주제로 즉시 블로그 글을 발행합니다.
* **📊 키워드 검색량 조회 (PRO)**: 이 키워드가 돈이 되는 황금 키워드인지 월간 검색량을 분석해 줍니다.
* **⚡ 경쟁 블로그 분석 (PRO)**: 해당 키워드로 상위 노출 중인 경쟁자들의 블로그를 분석하여 인사이트를 제공합니다.

### 🌐 5. 기타 일반 웹페이지
쇼핑몰, 일반 회사 홈페이지 등을 보고 있었다면?
* **🔖 URL 아이디어 금고에 저장**: 나중에 보기 위해 주소를 아이디어 금고에 스크랩합니다.
* **🖋️ 페이지 내용으로 글쓰기 (PRO)**: 해당 페이지에 있는 텍스트들을 긁어와서, 제품 소개글이나 리뷰 글처럼 자연스러운 포스팅으로 변환해 줍니다.

---

## ⚡ 사용 꿀팁 (Workflow)

1. 웹서핑을 하다가 글감으로 쓰기 좋은 **유튜브나 뉴스 기사**를 발견한다.
2. 탭을 닫지 않고, 켜져 있는 **MAZA Studio 탭을 클릭**하여 돌아간다.
3. 화면 오른쪽에 스르륵 나타나는 **MAZA Bridge 패널**에서 원하는 글쓰기 버튼을 누른다.
4. AI가 알아서 초안을 완성하고 티스토리에 발행할 때까지 커피를 마신다! ☕

이제 복사 + 붙여넣기의 늪에서 벗어나세요. 눈으로 보고, 클릭 한 번으로 나만의 콘텐츠를 생산해 보세요!

---

## ⚙️ 브릿지 및 발행 원리

MAZA Bridge가 수집한 컨텍스트를 바탕으로 AI가 글을 작성하고 나면, 최종적으로 사용자의 블로그에 글이 올라가는 과정은 Maza Extension의 **DOM 인젝션(DOM Injection)** 기능을 통해 자동으로 이루어집니다. 
어떻게 자동으로 내 블로그에 글이 발행되는지 궁금하시다면, [Maza Extension 기술 가이드](./EXTENSION_GUIDE.md) 문서를 참고해 주세요!
