# 📱 MAZA Studio 모바일 홈 화면 설치 가이드 (100% 활용법)

MAZA Studio는 모바일 환경에서도 완벽하게 작동하도록 설계된 PWA(Progressive Web App) 기술을 지원합니다.
스마트폰 '홈 화면에 추가' 기능을 활용하면, 앱 스토어에서 번거롭게 다운로드할 필요 없이 **마치 네이티브 모바일 앱처럼 전체 화면으로 빠르고 쾌적하게 사용**하실 수 있습니다.

특히 스마트폰으로 찍은 사진을 바로 올려서 포스팅하는 **[비전 라이터(Vision Writer)]** 기능을 200% 활용하기 위해서는 모바일 홈 화면 설치가 필수적입니다!

아래 기종별 가이드를 따라 1분 만에 설치를 완료해 보세요.

---

## 🍎 1. iPhone (iOS) 설치 가이드

아이폰에서는 반드시 기본 브라우저인 **Safari(사파리)** 앱을 사용하셔야 홈 화면에 완벽한 앱 형태로 설치됩니다. (크롬이나 네이버 앱에서는 기능이 제한될 수 있습니다.)

### 📌 설치 순서
1. **Safari 브라우저 실행** 
   * 아이폰에서 `Safari` 앱을 엽니다.
2. **MAZA Studio 접속** 
   * 주소창에 `mazastudio.kr` (또는 사용자님의 고유 주소)을 입력하여 마자스튜디오에 접속합니다.
3. **공유하기 버튼 터치** 
   * 화면 하단 정중앙에 있는 **[공유 아이콘]**(네모 박스 위로 화살표가 솟아오른 모양 📤)을 터치합니다.
4. **'홈 화면에 추가' 선택** 
   * 위로 올라온 메뉴 창을 아래로 조금 스크롤하여 **[홈 화면에 추가]** (또는 'Add to Home Screen') 메뉴를 찾아 터치합니다.
5. **설치 완료** 
   * 우측 상단의 **[추가]** 버튼을 누릅니다.
   * 홈 화면으로 튕겨져 나가면서, 예쁜 MAZA Studio 앱 아이콘이 홈 화면에 생성된 것을 확인할 수 있습니다!

> 💡 **Tip:** 생성된 아이콘을 꾹 눌러 자주 사용하는 폴더(예: '블로그 관리', '업무')에 넣어두시면 더욱 편리합니다.

## 🤖 2. Galaxy (Android) 설치 가이드

안드로이드 스마트폰(삼성 갤럭시 등)에서는 **Chrome(크롬)** 브라우저 또는 **삼성 인터넷** 브라우저를 통해 간단히 설치할 수 있습니다.

### 📌 A. Chrome (크롬) 브라우저를 사용할 경우
1. **Chrome 앱 실행 후 접속** 
   * 크롬 브라우저를 열고 `mazastudio.kr` 에 접속합니다.
2. **메뉴 버튼 터치** 
   * 화면 우측 상단에 있는 **[세로 점 3개(⋮)]** 메뉴 아이콘을 터치합니다.
3. **'홈 화면에 추가' 선택** 
   * 열린 메뉴 리스트에서 **[홈 화면에 추가]** (또는 '앱 설치') 버튼을 터치합니다.
4. **확인 및 설치** 
   * 나타나는 팝업창에서 **[추가]**를 누르고, 아이콘 크기를 확인하는 창이 뜨면 한 번 더 **[추가]**를 누릅니다.
   * 바탕화면에 MAZA Studio 앱 아이콘이 생성됩니다.

### 📌 B. 삼성 인터넷 브라우저를 사용할 경우
1. **삼성 인터넷 실행 후 접속** 
   * 기본 브라우저인 삼성 인터넷을 열고 `mazastudio.kr` 에 접속합니다.
2. **메뉴 버튼 터치** 
   * 화면 우측 하단에 있는 **[가로 줄 3개(≡)]** 메뉴 아이콘을 터치합니다.
3. **'현재 페이지 추가' 선택** 
   * 메뉴 중 **[+ 현재 페이지 추가]** 항목을 터치합니다.
4. **'홈 화면' 선택** 
   * 추가할 위치를 묻는 창에서 **[홈 화면]**을 선택하고 **[추가]**를 누르면 완료됩니다.

## ✨ 홈 화면에 앱으로 설치하면 무엇이 좋나요?

웹사이트를 북마크해두는 것과 **'홈 화면에 추가(PWA)'** 하는 것은 차원이 다른 경험을 제공합니다!

1. **🚀 압도적인 속도와 몰입감 (풀스크린)** 
   * 브라우저의 지저분한 주소창과 하단 메뉴바가 모두 사라지고, 화면을 100% 꽉 채우는 진짜 모바일 앱처럼 실행됩니다.
2. **📸 비전 라이터 (Vision Writer) 최적화** 
   * 식당, 카페, 여행지에서 사진을 찍은 직후, 바탕화면의 Maza 앱을 켜서 사진을 업로드하면 곧바로 퀄리티 높은 실전 리뷰 포스팅이 완성됩니다.
3. **🔋 배터리 및 데이터 절약** 
   * 일반 웹서핑보다 훨씬 가볍고 빠르게 로딩되도록 캐시(Cache)가 최적화되어 동작합니다.
4. **🔒 로그인 상태 유지** 
   * 매번 로그인할 필요 없이 생체 인식(Face ID / 지문)이나 토큰을 통해 빠르고 안전하게 접속 상태가 유지됩니다.
