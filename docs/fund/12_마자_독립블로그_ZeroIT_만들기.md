# 🚀 마자 독립 블로그 (Zero-IT) 원클릭 세팅 가이드

마자 스튜디오의 'Zero-IT' 기술을 활용하면, 코딩이나 복잡한 인프라(서버) 지식 없이도 완벽한 구글 애드센스 최적화 블로그를 단 3분 만에 생성할 수 있습니다. 

과거처럼 깃허브(GitHub) 가입, 넷리파이(Netlify) 팝업 연동 같은 귀찮은 과정을 거치지 마세요. 회원님은 도메인만 입력하시고 나머지는 마자가 전부 알아서 해드립니다!

---

## 🛠️ 블로그 자동화 마법사 진행 순서

### Step 1. 도메인 및 기본 정보 입력
- **[내 사이트]** 메뉴에서 **[+ 새 사이트 추가]** 버튼을 눌러 마법사를 시작합니다.
- **플랫폼 선택**: `마자 독립 블로그 (Zero-IT)`를 선택합니다.
- **도메인 입력**: 가비아, 호스팅케이알 등에서 구매하신 본인의 도메인 주소(예: `mazapost.com`)를 입력합니다.
- **블로그 정보**: 블로그 이름, 운영 주체명, 연락처 이메일을 입력합니다. (이 정보는 추후 블로그의 약관, 개인정보처리방침 페이지에 예쁘게 자동 삽입됩니다.)
- **발행 간격 설정**: 어뷰징 방지를 위해 하루 몇 개의 글을 텀을 두고 올릴지 간격을 설정합니다.

### Step 2. 블루프린트(테마) 선택
- 어떤 수익형 주제로 블로그를 운영할지(예: IT, 건강, 대출, 정부지원금 등) 검증된 갤러리에서 **블루프린트**를 선택합니다. 

### Step 3. 대량 포스팅 일괄 생성 가동 🌟
- 앞서 고른 테마로 생성할 **초기 글의 개수(예: 15개)**를 설정하고 대량 생산을 시작합니다.
- 💡 **[비밀 무기: Backdating]**: AI가 글을 쓸 때 오늘 날짜로만 15개를 올리는 것이 아닙니다. 구글 SEO 최적화와 애드센스 승인을 위해 **'3일 전 5개, 2일 전 5개, 오늘 5개'** 식으로 과거 날짜로 백데이팅하여 글을 배치합니다. 

### Step 4. 구글 계정 연동
- 백그라운드에서 AI가 15개의 글을 열심히 쓰는 동안, 회원님은 화면의 안내에 따라 **구글 계정으로 로그인**합니다. 
- 시스템이 블로그 검색 노출을 위한 **구글 서치콘솔**과 방문자 통계를 위한 **GA4(구글 애널리틱스)** 권한을 안전하게 받아옵니다.

### Step 5. [Zero-IT 자동 구축] 및 배포 🌟
- 구글 연동이 끝나면 마법이 시작됩니다. 유저가 깃허브나 넷리파이 팝업을 클릭하지 않아도, 시스템이 백그라운드에서 **GitHub 저장소를 만들고 Netlify 무료 호스팅 서버를 100% 자동 구축**합니다.
- 동시에 앞서 받아온 구글 추적 코드(메타데이터)와 완성된 15개의 블로그 글을 서버에 밀어넣어 전 세계로 **최종 배포(Build)** 합니다.

### Step 6. 최종 핑(Ping) 전송 및 애드센스 신청
- 서버 배포가 완료되면, 시스템이 구글 검색엔진에 핑(Ping)을 날려 "새로운 글이 15개나 있으니 수집해가라"고 호출합니다.
- 이제 마법사가 종료됩니다. 완성된 내 도메인을 들고 **구글 애드센스에 승인 심사를 신청**하세요!

## 🌐 단 하나! 외부에서 해야 할 일 (DNS 설정)

위 6단계가 마자 스튜디오 안에서 3분 만에 자동으로 끝납니다. 회원님은 딱 한 번, 도메인을 구매하신 사이트(가비아 등)에 들어가서 도메인에 이름표를 달아주기만 하면 됩니다.

1. **도메인 구입처 접속 ➡️ DNS 레코드 설정 메뉴**
2. **A 레코드 추가:** IP 주소 란에 Netlify 고정 주소인 `75.2.60.5`를 입력하고 저장합니다.

> 👉 상세한 입력 방법은 `05_도메인_연결_가이드.md` 문서를 참고하세요!
