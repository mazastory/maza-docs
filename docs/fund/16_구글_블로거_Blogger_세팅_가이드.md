# 🟠 구글 블로거 (Blogger/Blogspot) 원클릭 세팅 가이드

구글 블로거(Blogspot)는 구글이 직접 운영하는 플랫폼으로, 서버 비용이 완전 무료이며 애드센스 연동이 가장 자연스럽고 강력하다는 장점이 있습니다. 
마자 스튜디오는 **Blogger Native API**를 완벽하게 연동하여 에러 없는 초고속 오토파일럿 자동화를 제공합니다!

---

## 🛠️ 구글 블로거 자동화 마법사 진행 순서

### Step 1. 도메인 및 기본 정보 입력
- **[내 사이트]** 메뉴에서 **[+ 새 사이트 추가]** 버튼을 눌러 마법사를 시작합니다.
- **플랫폼 선택**: `구글 블로거 (Blogger)`를 선택합니다.
- **도메인 입력**: 구글 블로거의 주소(예: `myblog.blogspot.com`) 또는 연결된 2차 도메인을 입력합니다.
- **발행 간격 설정**: 하루에 몇 개의 글을 텀을 두고 올릴지 간격을 설정합니다.

### Step 2. 구글 계정 API 다이렉트 연동 🌟
- 블로거는 구글의 서비스이므로, 버튼 클릭 한 번으로 **구글 계정 연동(OAuth)**을 진행합니다.
- 로그인 즉시 회원님의 블로거 계정과 마자 스튜디오가 다이렉트로 연결되며, 어떤 블로그에 글을 쓸지(Blog ID) 자동으로 스캔하여 선택할 수 있게 해줍니다.

### Step 3. 블루프린트(테마) 선택
- 어떤 수익형 주제로 블로그를 운영할지 갤러리에서 **블루프린트**를 선택합니다.

### Step 4. 대량 포스팅 일괄 생성 가동 🌟
- 생성할 **초기 글의 개수(예: 15개)**를 설정하고 대량 생산을 시작합니다.
- 💡 **[비밀 무기: Backdating]**: AI가 글을 쓸 때 오늘 날짜로만 15개를 올리는 것이 아닙니다. 구글 SEO 최적화와 애드센스 승인을 위해 **'3일 전 5개, 2일 전 5개, 오늘 5개'** 식으로 과거 날짜로 백데이팅하여 구글 블로거에 다이렉트로 주입합니다. API 연동이기 때문에 속도가 1초도 걸리지 않습니다!

### Step 5. 서치콘솔 및 애널리틱스(GA4) 연동
- 블로거 연동 시 사용한 구글 계정을 통해, 검색 노출을 위한 **구글 서치콘솔**과 **GA4(구글 애널리틱스)** 권한까지 원스톱으로 세팅됩니다. 다른 플랫폼처럼 메타 태그를 수동으로 복사해서 붙여넣을 필요가 전혀 없습니다!

### Step 6. 최종 핑(Ping) 전송 및 애드센스 신청
- 세팅이 완료되면 시스템이 구글 검색엔진에 핑(Ping)을 날려 글 수집을 요청합니다.
- 구글 블로거 관리자 메뉴의 [수익] 탭을 눌러 클릭 한 번으로 **구글 애드센스에 승인 심사를 신청**하세요!
