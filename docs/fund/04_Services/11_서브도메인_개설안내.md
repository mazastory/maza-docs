# 서브도메인 개설 (Maza Zero)

**Maza Zero**는 도메인, 호스팅, 설치 없이 마자가 직접 운영하는 블로그를 즉시 개설해 드리는 서비스입니다. 현재 준비 중입니다.

## 🕒 준비 중 (Beta 예정)
Maza Zero 서비스는 현재 내부 개발 중입니다. 출시 알림을 받으시려면 마자 스튜디오 대시보드에서 [알림 신청] 버튼을 눌러주세요.

## Maza Zero란?
- 🌐 **즉시 개설**: 도메인 구매 없이 `.mazastory.com` 서브도메인으로 블로그가 즉시 생성됩니다.
- ⚡ **Zero 설정**: 워드프레스 설치, 테마 선택, 플러그인 설정이 전혀 필요 없습니다.
- 📦 **완전 자동화**: 개설 즉시 Maza 오토파일럿과 연결되어 바로 글 발행이 시작됩니다.

## 지금 당장 시작하려면?
Maza Zero를 기다리는 동안, 아래 방법 중 하나로 즉시 시작할 수 있습니다.
- **티스토리**: 무료, 한국어 SEO에 최적화, 가장 빠른 시작 (추천)
- **블로그스팟**: 구글 직영, 애드센스 연동 최고 속도 (빠른 승인)
- **워드프레스**: 최고의 커스터마이징, 별도 호스팅 필요 (고급)
