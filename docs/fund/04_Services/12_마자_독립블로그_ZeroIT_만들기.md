# 🚀 마자 독립 블로그 (Zero-IT) 원클릭 세팅 가이드

마자 스튜디오의 'Zero-IT' 기술을 활용하면, 코딩이나 복잡한 인프라(서버) 지식 없이도 완벽한 구글 애드센스 최적화 블로그를 단 3분 만에 생성할 수 있습니다. 

단 한 번(최초 1회 진입 시)만 **넷리파이(Netlify)**를 팝업으로 연동해두면, **이후 2번째 블로그부터는 번거로운 연동 과정을 완전히 생략**합니다! 회원님은 도메인만 입력하시고 [배포] 버튼을 누르면 나머지는 마자 백엔드가 전부 알아서 해드립니다. 복잡한 깃허브(GitHub) 연동은 최초 3개월간 쳐다보실 필요도 없습니다.

---

## 🛠️ 블로그 자동화 마법사 진행 순서

### Step 1. 도메인 입력 및 인프라 선구축 🌟
- **[내 사이트]** 메뉴에서 **[+ 새 사이트 추가]** 버튼을 눌러 마법사를 시작합니다.
- **플랫폼 선택**: `마자 독립 블로그 (Zero-IT)`를 선택합니다.
- **조건부 인프라 연동**: 최초 1회에 한해 **Netlify** 연동 버튼을 눌러 서버 권한을 허용합니다. (초기 세팅 시 Git 연동은 불필요합니다)
- **도메인 입력**: 구매하신 도메인 주소를 입력하고 [다음]을 누르는 순간, 백엔드에서 **즉시 회원님의 Netlify에 전용 서버를 구축**합니다.
- **DNS 안내 모달**: 서버가 생성되자마자 화면에 Netlify IP(`75.2.60.5`)가 뜹니다. 지금 바로 가비아 등에 가셔서 A 레코드를 설정해 두세요!

### Step 2. 구글 계정 연동 및 메타데이터 획득
- 화면의 안내에 따라 **구글 계정으로 로그인**합니다. 
- 시스템이 블로그 검색 노출을 위한 **구글 서치콘솔**과 방문자 통계를 위한 **GA4(구글 애널리틱스)** 권한을 안전하게 받아옵니다.

### Step 3. 블루프린트(테마) 선택
- 어떤 수익형 주제로 블로그를 운영할지(예: IT, 건강, 정부지원금 등) 검증된 갤러리에서 **블루프린트**를 선택합니다. 

### Step 4. 대량 포스팅 생성 및 DNS 숙성 대기 🌟
- 테마에 맞는 **초기 글(예: 15개)** 생성을 시작합니다.
- 💡 **[핵심 원리]**: AI가 15개의 글을 작성하는 데에는 약 10~20분의 물리적 시간이 소요됩니다. **이 대기 시간 동안 앞서 설정한 도메인 DNS와 SSL 인증서가 전 세계망에 완벽하게 전파(Propagation)됩니다.** 유저는 기다리는 동안 다른 블로그를 하나 더 세팅하고 오셔도 좋습니다!

### Step 5. 최종 콘텐츠 주입(Inject) 및 배포
- 목표한 15개의 글 생성이 모두 끝나면, 마법사 하단의 **[최종 배포 및 핑 전송] 버튼의 락(Lock)이 풀립니다.**
- 버튼을 누르면, 이미 20분 동안 튼튼하게 안정화된 Netlify 서버에 방금 쓴 15개의 글과 구글 메타데이터를 쏙 밀어넣어(Inject) 최종 배포(Build)합니다.

### Step 6. 최종 핑(Ping) 전송 및 애드센스 신청
- 서버 빌드가 완료되면, 시스템이 즉각 구글 검색엔진에 핑(Ping)을 날려 "새로운 글이 15개나 있으니 수집해가라"고 호출합니다. 인프라가 완벽히 굳어진 상태라 에러 확률은 0%에 수렴합니다.
- 이제 마법사가 종료됩니다. 완성된 내 도메인을 들고 **구글 애드센스에 승인 심사를 신청**하세요!

## 🌐 단 하나! 외부에서 해야 할 일 (DNS 설정)

위 6단계가 마자 스튜디오 안에서 3분 만에 자동으로 끝납니다. 회원님은 딱 한 번, 도메인을 구매하신 사이트(가비아 등)에 들어가서 도메인에 이름표를 달아주기만 하면 됩니다.

1. **도메인 구입처 접속 ➡️ DNS 레코드 설정 메뉴**
2. **A 레코드 추가:** IP 주소 란에 Netlify 고정 주소인 `75.2.60.5`를 입력하고 저장합니다.

> 👉 상세한 입력 방법은 `05_도메인_연결_가이드.md` 문서를 참고하세요!

## 🎓 [비전] 3개월 후: 마자 스튜디오 졸업 및 완전 독립

마자 스튜디오의 철학은 '고객을 영원히 플랫폼에 종속시키는 것'이 아닙니다. 가장 이탈률이 높은 마의 3개월 구간을 MAZA 오토파일럿으로 안전하게 넘기고 달러 수익이 발생하기 시작하면, 회원님은 **완벽한 '디지털 건물주'로 독립**할 수 있습니다.

### 🎁 졸업 시 무상 제공 패키지 (와디즈 VIP 한정 엑시트 전략)
가입 후 3개월이 경과하면 대시보드의 **[독립(이관) 신청]** 메뉴가 활성화됩니다. 이번 와디즈 펀딩 서포터분들께는 정식 오픈 시 **50만 원 상당에 판매될 독립 라이선스 패키지**를 조건 없이 영구 무상 양도해 드립니다.

1. **블로그 프론트엔드 소스코드**: 0.1초 로딩 속도와 **'글로벌 다국어 자동 번역 언어팩'**이 완벽히 탑재된 Maza Zero-IT 원본 코드 
2. **글 데이터 (DB)**: 그동안 발행한 수천 개의 포스팅 데이터를 엑셀(CSV)로 온전히 추출
3. **개인용 어드민(CMS) 패널**: 더 이상 마자 스튜디오에 로그인하지 않아도, 회원님 본인만 접속해서 글을 쓰고 블로그를 관리할 수 있는 독립형 어드민 페이지

이관 신청 시, 멈춰 있던 깃허브(GitHub) 연동 창이 활성화되며 클릭 한 번에 모든 코드와 데이터가 회원님의 개인 소유로 완벽히 이전됩니다. **MAZA는 종속이 아닌, 완벽한 자립을 위한 3개월의 부스터입니다.**
