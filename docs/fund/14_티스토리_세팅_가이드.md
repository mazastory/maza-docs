# 🟠 티스토리(Tistory) 원클릭 세팅 가이드

마자 스튜디오는 티스토리 블로그를 위한 완벽한 RPA(Robotic Process Automation) 연동을 지원합니다. 
티스토리는 2024년부터 전통적인 Open API 키 발급을 중단했지만 걱정하지 마세요. 마자 스튜디오는 **'크롬 익스텐션(Chrome Extension)'을 통한 API-Free 주입 방식**으로 티스토리 자동화를 완벽히 구현했습니다!

---

## 🛠️ 티스토리 자동화 마법사 진행 순서

### Step 1. 도메인 및 기본 정보 입력
- **[내 사이트]** 메뉴에서 **[+ 새 사이트 추가]** 버튼을 눌러 마법사를 시작합니다.
- **플랫폼 선택**: `티스토리 (Tistory)`를 선택합니다.
- **블로그 주소 입력**: 본인의 티스토리 주소(예: `myblog.tistory.com`) 또는 연결된 개인 도메인을 입력합니다.
- **발행 간격 설정**: 티스토리의 엄격한 어뷰징 제재를 피하기 위해, 하루 몇 개의 글을 텀을 두고 올릴지 간격을 설정합니다. (스피드 모드 / 안전 모드)

### Step 2. 블루프린트(테마) 선택
- 어떤 수익형 주제로 블로그를 운영할지 갤러리에서 **블루프린트**를 선택합니다.

### Step 3. 대량 포스팅 일괄 생성 가동 🌟
- 생성할 **초기 글의 개수(예: 15개)**를 설정하고 대량 생산을 시작합니다.
- 💡 **[비밀 무기: Backdating]**: AI가 글을 쓸 때 오늘 날짜로만 15개를 올리는 것이 아닙니다. 구글 SEO 최적화와 애드센스 승인을 위해 **'3일 전 5개, 2일 전 5개, 오늘 5개'** 식으로 과거 날짜로 백데이팅하여 티스토리 예약 큐(Queue)에 배치합니다. 

### Step 4. 마자 크롬 익스텐션 설치 (티스토리 필수!) 🌟
- 티스토리는 외부 API를 막아두었기 때문에, 회원님의 브라우저가 직접 글을 발행해 주어야 합니다.
- 화면의 안내에 따라 **[Maza Studio Chrome Extension]**을 설치합니다.
- 티스토리에 로그인만 해두시면, 익스텐션이 AI가 쓴 글을 10분~수 시간 간격으로 아주 자연스럽게(사람이 직접 쓰는 것처럼) 티스토리 에디터에 주입합니다!

### Step 5. 구글 계정 연동 및 메타데이터 주입
- 글이 백그라운드에서 예약/생성되는 동안, 화면 안내에 따라 **구글 계정으로 로그인**합니다.
- 시스템이 블로그 검색 노출을 위한 **구글 서치콘솔**과 **GA4(구글 애널리틱스)** 권한을 받아오고, 티스토리 스킨 편집에 필요한 메타 태그를 제공합니다.

### Step 6. 최종 핑(Ping) 전송 및 애드센스 신청
- 세팅이 완료되면 시스템이 구글 검색엔진에 핑(Ping)을 날려 글 수집을 요청합니다.
- 이제 티스토리 도메인으로 **구글 애드센스에 승인 심사를 신청**하세요!
