# MAZA Studio 제품 소개서

## 1. 서비스 개요
**시간과 체력의 한계를 넘어선 'ZERO-IT' 글로벌 수익 파이프라인**
MAZA Studio는 코딩, 글쓰기, 서버 관리 없이 AI가 최대 10개 언어로 블로그 콘텐츠를 자동 발행해 주는 **'올인원 블로그 수익화 자동화 플랫폼'**입니다. 

온라인 꿀수입을 꿈꾸지만 복잡한 IT 설정, 매일 글쓰기의 압박, 까다로운 애드센스 승인 단계에서 중도 포기했던 분들에게 MAZA가 가장 현실적인 대안을 제시합니다.

## 2. 핵심 가치 및 특장점 (The MAZA Solution)

* **ZERO-IT (기술적 진입장벽 제거)**
  * 코딩, 서버, SEO에 대한 학습이 전혀 필요 없습니다. 오직 콘텐츠의 방향에만 집중하세요.
  * 구글 Core Web Vitals 및 구조화 데이터 기반의 기술적 SEO 최적화가 자동 적용됩니다.
* **ZERO-WORK (스케줄 기반 자동 발행)**
  * AI가 키워드와 주제를 스스로 탐색하고 다국어 콘텐츠를 작성합니다.
  * 예약된 스케줄에 맞춰 자동으로 콘텐츠가 발행되므로, 매일 컴퓨터 앞에 앉아있을 필요가 없습니다.
* **ZERO-LIMIT (글로벌 10개국 언어 동시 발행)**
  * 단일 언어(한국어) 대비 훨씬 넓은 글로벌 트래픽 풀을 확보합니다.
  * 영어, 일본어, 스페인어, 독일어, 프랑스어 등 클릭 단가(CPC)가 높은 글로벌 타겟 시장을 공략하여 수익 파이의 한계를 넓힙니다.

## 3. 이런 분들에게 추천합니다 (Target Persona)
"매일 컴퓨터 앞에 앉아 글을 쓰는 것"이 물리적으로 어려운 모든 분이 MAZA의 고객입니다.

* **바쁜 N잡러 / 직장인:** 퇴근 후 투잡 할 시간이 없으신 분 (잠든 사이 AI가 자동 발행)
* **건강 문제 / 장기 환자:** 체력적 한계로 지속적인 작업이 어려우신 분 (최소한의 클릭으로 파이프 구축)
* **시니어 / 은퇴자:** IT 용어와 설정이 장벽으로 느껴지시는 분 (코딩 0, 설정 0)
* **육아 중 전업주부:** 24시간 육아로 자기 시간이 없으신 분 (아이 재우고 클릭 한 번)
* **기타:** 부업 초보, 대학생/취준생, 소상공인/자영업자, 해외 교포 및 워홀러

## 4. 하나로 연결된 MAZA 생태계

1. **MAZA Studio (통합 컨트롤 타워):** 티스토리, 워드프레스, 구글 블로거 등 여러 플랫폼의 자산을 하나의 대시보드에서 오토파일럿으로 관리합니다.
2. **MAZA Zero-IT Blog (수익 최적화 최전선):** 기술적 SEO 최적화, 빠른 로딩 속도, 다국어 자동화가 기본 탑재된 독립형 블로그를 제공합니다.
3. **MAZA Academy (성공을 위한 지식 허브):** 초기 세팅부터 롱테일 키워드 발굴, 애드센스 활용 팁 등 실제 운영 사례를 안내합니다.

> **차별점: 애드센스 승인 준비 과정 자동화**
> 애드센스 승인에 필수적인 페이지(소개, 연락처, 개인정보처리방침 등) 생성과 초기 세팅을 자동화하여 반복적인 설정 작업 없이 빠르게 시작할 수 있습니다.

## 5. 워드프레스 vs MAZA Studio 비교

| 비교 항목 | 기존 워드프레스 (자가 운영) | **MAZA Studio** |
| --- | --- | --- |
| **초기 세팅** | 수일~수주 소요 (도메인, 호스팅 등) | **클릭 몇 번으로 수분 내 완성** |
| **월 운영 비용** | 호스팅 + 플러그인 비용 별도 누적 | **구독료 포함 올인원 (별도 서버비 없음)** |
| **다국어 비용** | 번역 플러그인 등 이중 과금 | **다국어 엔진 내장 (추가 비용 없음)** |
| **SEO 자동화** | 별도 플러그인 설치 및 복잡한 설정 | **기술적 SEO 요소 자동 적용** |
| **기술 요구도** | 중~고급 (코딩/서버 기본 지식 필요) | **코딩 지식 없이 즉시 사용 가능** |

## 6. 종속 없는 투명한 구조 (Exit Strategy)
플랫폼에 종속될까 걱정하지 마세요. MAZA는 초기 부스터 역할을 하며, 운영이 안정화되어 독립을 원하실 때 **'독립 이관 패키지'**를 제공합니다.

* **프론트엔드 소스코드:** 다국어 언어팩이 포함된 블로그 원본 코드 제공
* **콘텐츠 데이터(DB):** 발행된 모든 포스팅 데이터를 CSV로 완전 추출 보장
* **개인용 어드민(CMS) 패널:** MAZA 없이 혼자서도 관리할 수 있는 독립형 패널 제공
