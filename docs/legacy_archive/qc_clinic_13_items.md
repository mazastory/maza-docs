# MAZA Studio QC Clinic : 13가지 심층 검사 항목

MAZA Studio의 **QC Clinic (Quality Control)** 시스템은 구글봇(Googlebot)의 크롤링 최적화와 AdSense 승인 통과율 극대화를 위해, 생성된 블로그를 대상으로 다음 13가지 핵심 인프라 및 콘텐츠 품질 항목을 전수 조사합니다.

이 검사는 스크립트 실행 한 번으로 자동 수행되며, 모든 항목에 대한 종합 등급(A+/A/B+/B/C/F)을 산출합니다.

---

## 🛠 1. 코어 인프라 및 성능 (Core Infrastructure & Performance)

### 1) ⚡ 서버 응답 속도 (Latency)
- **점검 내용:** 메인 홈페이지의 초기 응답 속도 측정.
- **통과 기준:** HTTP 200 OK 상태 반환 및 3,000ms 이하 응답. (로딩 속도 저하 시 감점)

### 2) 💰 AdSense 스크립트 무결성
- **점검 내용:** HTML `<head>` 영역 내 `pagead2.googlesyndication.com` 스크립트 존재 여부 및 `ca-pub-` ID 정상 추출.
- **통과 기준:** 스크립트 필수 삽입 및 고유 Publisher ID 검출.

### 3) 🛡️ ads.txt 파일 정합성
- **점검 내용:** 루트 디렉토리 `/ads.txt` 파일 존재 여부 및 내부 포맷 유효성 검사.
- **통과 기준:** HTTP 200 정상 반환 및 HTML에 삽입된 AdSense Pub-ID와의 100% 일치 교차 검증.

---

## 📈 2. 트래킹 및 SEO 인증 (Tracking & SEO Verification)

### 4) 🔍 구글 서치콘솔 (Google Search Console)
- **점검 내용:** `google-site-verification` 소유권 확인 메타 태그 존재 여부.
- **통과 기준:** 태그 정상 발견 시 통과. (DNS 인증 방식 등은 예외 처리 가능)

### 5) 📊 구글 애널리틱스 (GA4)
- **점검 내용:** 웹 트래픽 분석을 위한 `gtag.js` 또는 `G-` 태그 존재 여부.
- **통과 기준:** 정적 HTML 내 태그 삽입 확인.

---

## 🧠 3. 심층 고급 SEO (Advanced Technical SEO)

### 6) 🔗 Open Graph (OG) 태그
- **점검 내용:** 소셜 미디어 공유 시 노출될 메타데이터 (`og:title`, `og:description`, `og:image`) 존재 여부.
- **통과 기준:** 3가지 핵심 OG 태그 모두 삽입 시 통과.

### 7) 📌 Canonical URL
- **점검 내용:** 중복 문서 패널티를 방지하기 위한 `<link rel="canonical">` 태그 적용 유무.
- **통과 기준:** 해당 태그 필수 포함.

### 8) 🧩 JSON-LD 구조화 데이터 (Schema Markup)
- **점검 내용:** 검색엔진 리치 스니펫(Rich Snippet)을 위한 `application/ld+json` 구조화 마크업 삽입 여부.
- **통과 기준:** 스키마 타입(`@type`) 정상 파싱 및 존재 확인.

### 9) 📡 RSS 피드 자동 동기화
- **점검 내용:** 검색엔진의 빠른 콘텐츠 색인을 위한 `/rss.xml` 활성화 및 HTML 헤더 내 선언 상태.
- **통과 기준:** RSS 피드 링크 삽입 및 접근 시 정상 XML 반환.

---

## ⚖️ 4. 신뢰성 및 콘텐츠 안전성 (E-E-A-T & Content Safety)

### 10) 🤝 필수 정책/신뢰성 페이지 (Trust Indicators)
- **점검 내용:** 구글 정책 준수를 위한 필수 페이지 링크(Privacy, Terms, About, Contact, Disclaimer 등) 존재 여부.
- **통과 기준:** 하단 Footer 영역 등에 최소 2개 이상의 필수 정책 페이지 링크 노출.

### 11) 🖼️ 포스트 이미지 과다/중복 주입 검사 (Image Clutter)
- **점검 내용:** 실제 발행된 개별 포스트 내부에 썸네일이나 불필요한 이미지가 과도하게 렌더링되었는지 스캔.
- **통과 기준:** 포스트 1개당 총 이미지 수 6장 이하 유지 (과도한 반복 및 로딩 지연 방어).

### 12) 🚨 기만성 콘텐츠 패턴 스캔 (Deceptive Safety)
- **점검 내용:** 본문 내 허위 자격 증명("의학박사", "CFA", "면허 보유" 등)이나 과장 광고 패턴이 존재하는지 검사.
- **통과 기준:** 애드센스 정책을 위반할 수 있는 의심 키워드 0건.

### 13) 🤖 기계적 AI 문구 스캔 (Cliche Scrubber)
- **점검 내용:** AI가 습관적으로 내뱉는 기계적 클리셰 ("오늘은~", "이번 포스팅에서는~", "알아보겠습니다") 남발 검사.
- **통과 기준:** 전체 본문에서 해당 클리셰 빈도수 3회 이하로 억제 (자연스러운 휴먼 터치 어투 유지).

---

## 🎯 QC Clinic 종합 등급 산출
위 13가지 항목을 바탕으로 100점 만점 기준의 점수가 산출되며, 다음과 같이 등급(Grade)이 매겨집니다:
- **A+** : 95점 이상 (Perfect, 즉시 승인 심사 가능)
- **A**  : 90점 ~ 94점 (우수)
- **B+** : 80점 ~ 89점 (양호, 일부 개선 권장)
- **B**  : 70점 ~ 79점 (보통)
- **C**  : 60점 ~ 69점 (개선 필요)
- **F**  : 60점 미만 (치명적 오류, 보완 후 재검사 필수)
