# 🎬 MAZA Studio 와디즈 프로모션 영상 스크립트
## "3단 분할 해방(Emancipation)의 미학"

**영상 길이:** 50초 ~ 1분 내외
**전체 톤앤매너:** 도입부 현실적·묵직함 ➔ 후반부 세련된 하이테크 따뜻함
**배포 채널:** 와디즈 상세페이지 최상단 + 페이스북·인스타그램·유튜브 광고

|---|---|---|---|

| **10~20초** | 화면이 매트 네이비 톤으로 전환. 스마트폰 또는 노트북에 MAZA 메인 대시보드가 스르륵 등장 | *"이제 기술을 배우지 마세요. AI가 당신의 디지털 대리인이 됩니다."* | 분위기 반전. 구원자로서의 MAZA 등장 |
| **20~40초** | ZERO-IT / ZERO-WORK / ZERO-LIMIT 카드 클로즈업 순차 등장 → 마우스가 [애드센스 자동 세팅] 클릭 → 프로그레스 바 차오름 → [완료] 화면 | *"클릭 단 3번. 애드센스 승인부터 전 세계 다국어 동시 발행까지 — 백엔드에서 완전 자동."* | **가장 중요한 구간.** 실제 UI 화면 녹화로 "진짜 작동한다"는 확신 도장 |
| **40~50초** | 침대에서 편안히 잠든 사람 / 병상·여가 중 스마트폰 화면에 달러(USD) 수익 현황판이 차오르는 장면 | *"당신이 잠든 사이에도, 본업에 집중하는 시간에도. 국경 없는 달러 자산이 알아서 차곡차곡."* | **방치형 무인 수익 시스템**이 주는 시공간의 자유를 시각화 |
| **50~60초** | 깔끔한 MAZA 로고 + 와디즈 오픈 알림 신청 버튼 화면 | *"디지털 소외 계층을 위한 반려 AI 수익 에이전트, MAZA. 지금 와디즈에서 만나보세요."* | CTA. 신뢰감 있는 톤으로 행동 유도 |

## 🎙️ 씬별 상세 나레이션 (풀 스크립트)

### 🔴 SCENE 1 — 도입부 "공감의 훅" (00~10초)
**[화면: 어두운 방, 머리 감싸 쥔 은퇴자·직장인 뒷모습. 모니터에 복잡한 영어 설정창]**

> **자막 (볼드체, 흰색):**
> "비싼 강의료 내고 노트북 앞에 앉았지만..."
> "용어조차 어려워 — 또 포기하셨나요?"

**[연출: 무거운 침묵 + 낮은 피아노 음 1~2음 정도만]**

---

### 🔵 SCENE 2 — 반전 "구원자 등장" (10~20초)
**[화면: 화면 전체가 매트 네이비(#0f172a)로 페이드인. MAZA 대시보드가 스르륵 등장]**

> **자막 (볼드체, 흰색 + 형광 강조):**
> "이제 기술을 배우지 마세요."
> "AI가 당신의 **디지털 대리인**이 됩니다."

**[연출: BGM이 세련된 신스 비트로 전환 — 따뜻하고 자신감 있는 분위기]**

---

### ✅ SCENE 3 — 핵심 증명 "ZERO-IT 3단 클릭" (20~40초)
**[화면: 카드 3장이 순차적으로 슬라이드 인]**

```
┌──────────────┐   ┌──────────────┐   ┌──────────────┐
│  ZERO-IT     │ → │  ZERO-WORK   │ → │  ZERO-LIMIT  │
│ 설치·세팅 없음 │   │  글쓰기 없음  │   │  한계 없는 수익│
└──────────────┘   └──────────────┘   └──────────────┘
```

**[화면: 마우스 커서가 [구글 애드센스 자동 세팅] 버튼을 딸깍 클릭]**
**[화면: 프로그레스 바가 0% → 100%로 차오르며 ✅ 완료 표시]**

> **자막:**
> "클릭 단 3번."
> "어려운 구글 애드센스 승인 신청부터"
> "전 세계 **다국어 동시 발행**까지"
> "백엔드에서 — **완전 자동 가동.**"

**[연출: 클릭 효과음(딸깍), 프로그레스 바 채워지는 소리, 완료 시 청량한 짧은 효과음]**

> [!IMPORTANT]
> 이 구간은 반드시 **실제 MAZA UI 화면 녹화본**을 사용할 것. 화려한 3D 그래픽보다 실제 작동 화면이 신뢰도 10배.

## 🛠️ 영상 제작 시 반드시 지켜야 할 3대 마케팅 법칙

### 법칙 1️⃣ — '기능 나열'이 아닌 '체감 결과'로 말하라

서포터들은 기술 용어를 보면 머리가 아파집니다.

| ❌ 나쁜 예 | ✅ 좋은 예 |
|---|---|
| "다국어 로직 API 탑재 및 동시 발행 기술" | "타자 한 번 안 쳤는데, 내 글이 미국·일본 구글에 동시 노출됩니다." |
| "GPT-4o 기반 E-E-A-T 최적화 자동화" | "글 500개가 쌓이면 — 자는 동안 하루 70만 원." |
| "크롬 익스텐션 에이전트 자동 포스팅" | "퇴근 후 켜두면, 내일 아침 10개 글이 올라가 있습니다." |

---

### 법칙 2️⃣ — 소리 없이 봐도 100% 이해되는 '자막 중심' 설계

> 와디즈 서포터 70% 이상은 **대중교통·직장에서 무음으로** 상세페이지 영상을 봅니다.

- 나레이션(보이스오버) 없이도 **볼드한 자막 + 화면 동작**만으로 스토리 완성
- 자막 폰트: **Noto Sans KR ExtraBold** / 최소 40pt 이상
- 클릭·완료 효과를 시각적으로 과장해서 표현 (마우스 클릭 이펙트 애니메이션)

---

### 법칙 3️⃣ — '실제 화면 녹화'가 어떤 그래픽보다 강력하다

> "진짜 완성되어 있구나" — 이 확신 한 방이 펀딩 전환율을 결정합니다.

- 화려한 3D 그래픽 소스 ❌
- MAZA 메인 대시보드에서 **실제 마우스가 움직이며 클릭하는 화면 녹화** ✅
- 실제 애드센스 수익 현황판 화면 (금액 일부 모자이크 후 사용) ✅
